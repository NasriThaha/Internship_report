# Internship_report

This repository contains my internship report covering hands-on training in core bioinformatics techniques and computational biology workflows.

Overview

The internship involved practical exposure to multiple domains including gene annotation, phylogenetics, sequence analysis, molecular docking, heatmap visualization, and homology modeling.

 Key Highlights

- Gene annotation and functional analysis of human proteins  
- Phylogenetic analysis of viral components  
- Sequence analysis and BLAST interpretation  
- Molecular docking studies on ACE2 protein  
- Heatmap-based gene expression analysis  
- Homology modeling of SARS-CoV-2 related proteins  

Tools & Softwares

- NCBI BLAST  
- R (data analysis & visualization)  
- Molecular docking tools  
- Public databases and resources
